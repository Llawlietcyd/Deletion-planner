# 5-Minute Demo Walkthrough Script

## 0:00 - 0:30 Problem framing

- People over-plan, under-execute, then carry task guilt.
- This product applies a deletion-first planning philosophy.

## 0:30 - 1:30 Inbox flow

- Open `Inbox`.
- Add a few tasks with different categories and priorities.
- Show that category and status are persisted and visible immediately.

## 1:30 - 2:45 Today execution flow

- Open `Today`.
- Generate a daily plan.
- Show AI reasoning and overload warning.
- Use the action panel to complete, defer, and delete tasks in one screen.

## 2:45 - 3:45 Review flow

- Open `Review`.
- Show deletion suggestions and historical decision trace.
- Demonstrate soft-delete and permanent-delete behavior.

## 3:45 - 4:30 Insights flow

- Open `Insights`.
- Explain completion rate and plan quality metrics.

## 4:30 - 5:00 Technical credibility

- Mention FastAPI v2 endpoints with schema validation.
- Mention PostgreSQL-ready setup + Alembic migration.
- Mention automated tests and Docker Compose staging setup.
